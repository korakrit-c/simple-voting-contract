## Setup

1. Clone Repository

    ```sh
    $ git clone https://github.com/korakrit-c/simple-voting-contract
    $ cd simple-voting-contract/dapp
    ```

2. Install Dependencies

    ```sh
    $ npm install
    ```

3. Run Dapp

    ```sh
    $ npm run start
    ```
